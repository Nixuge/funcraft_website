Qu'est ce site?

Ce site est une copie quasi identique du site de Funcraft, serveur minecraft fermé le 26 août 2023.


A quoi puis-je accéder sur ce site?

Tout le site principal hors compte (y compris vos statistiques), tous les threads + les posts du forum (en dev). Malheureusement, je n'ai pas eu le temps de prendre les profils du forum.


Est-ce que ce site est geré par un administrateur/staff funcraft?

Non.


Les classements sont différents de ceux sur la Wayback Machine/que j'ai sauvegardés

Contrairement au site de base, les comptes bannis et les comptes inactifs depuis plus de 2 mois sont comptés dans le classement.
Je ne souhaite pas changer ce comportement, même si il nuit techniquement à la parité avec le site original.
J'ai également ajouté les classements pour tous les mois disponibles de chaque mode de jeu.


Les maps de Funcraft sont-elles disponibles?

Oui, sur funcraftbackup.nixuge.me
